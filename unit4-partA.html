Q.1 Define CNF form.

Every Production is in the form N.T->N.T N.T or N.T->T where N.T stands for Non Terminal and T stands for Terminal

Q.2 Eliminate the ε production from the CFG given below
A-> 0 B 1 | 1 B 1
B-> 0 B | 1 B | ε

Eliminate the ε production from the CFG given below

B-> 0 B | 1 B | ε

Q.3 Define GNF Form

Every Production is in the form of N.T->N.T α or N.T->T where α stands for any number of Terminal Symbols

Symbol C is non-generative, after removing productions with C we have the below grammar,

Q.5 What are the three ways to simplify a context free grammar?

By removing the useless symbols from the set of productions.
By eliminating the empty productions.
By eliminating the unit productions.

Q.6 What do you mean by null production and unit production? Give an example.

In a grammar any of the Production body has ε then that particular production is called Null Production. In a grammar any of the production body has Single Non Terminal then that particular production is called Unit Production

Q.10 What is a multitape turing machine?

A Multitape Turing machine has a finite control with some finite number of tapes. Each tape is divided into cells and each cell can hold any symbol of the finite tape alphabet. The set of tape symbol includes a blank, and has a subset called the input symbols, of which the blank is not a member.

Q.1 Design a TM that accepts the language of odd integers written in binary.

To accept odd valued binary strings, we only have to look at the last bit. The TM moves right until it reads a blank, moves left one space and acepts if and only if there is a 1 on the tape

Q.11 What is a multiple track Turing machine?

A TM in which the input tape is divided into multiple tracks where each track is having different inputs is called multiple track TM.

Q.12 Define Halting Problem

The halting problem is the problem of determining, from a description of an arbitrary computer program and an input, whether the program will finish running or continue to run forever

Q.18 Is Halting problem decidable or undecidable problem

The machine starting at given configuration will halt after finite number of steps or will never reach to a halt state. For a given configuration and input tape we can't determine whether the machine will ever halt. This problem is called halting problem. The halting problem is unsolvable. Hence it is undecidable problem.

Q.21 Is it possible that a turing machine could be considered as a computer of functions from integers to integers? If yes justify

Yes, TMs simulate computer of functions from integers to integers. That means it is a device for computing integer valued functions. In this scheme integers were presented in unary as blocks of a single character and machine computed by changing the lengths of blocks or by constructing new blocks on the input tape

Q.22 What does Multitape turing machine contain?

1. The input, a finite sequence of input symbols, is placed on the first tape.
2. All other cells of all the tapes hold the blank.
3. The final control is in initial state.
4. The head of the first tape is at the left end of input.
5. All other tape heads are at some arbitrary cell.

Q.23 Define Subroutines in Turing machine

A Turing machine subroutine is a set of states that performs some useful process. This set of states includes a start state and another state that temporarily has no moves, and that serves at the return state to pass control to whatever other set of states called the subroutine
